# Proposed commit messages

Run these to land the change:

```bash
git add .
git commit -m "Add new feature."
git commit -m "feat(  user): add login flow"
git commit -m "feet: typo in type"
git commit -m "feat:no space after colon"
git commit -m "fix!(api): bang in wrong place"
git commit -m "chore(): empty scope"
```

Or write a longer one in a commit-message file:

```gitcommit
Refactor: simplify the loader!

Detailed body explaining why.
```
