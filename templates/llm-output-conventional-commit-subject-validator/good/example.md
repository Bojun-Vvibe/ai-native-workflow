# Proposed commit messages

Run these to land the change:

```bash
git add .
git commit -m "feat(auth): add login flow"
git commit -m "fix(loader): handle empty input"
git commit -m "refactor!: drop deprecated v1 endpoint"
```

Or write a longer one in a commit-message file:

```gitcommit
docs(readme): document the new flag

Detailed body explaining why.
```
