# Good example — continuous blockquotes

The quote below uses a bare `>` marker to keep both paragraphs inside the
same blockquote:

> First paragraph of the quote, which sets up the topic.
>
> Second paragraph as a proper continuation, rendered as one blockquote.

A second well-formed quote:

> Authors who remember the bare `>` marker
>
> get a single continuous blockquote with two paragraphs.

A code fence with quoted-looking lines inside is still ignored:

```
> sample one

> sample two
```

End of file.
