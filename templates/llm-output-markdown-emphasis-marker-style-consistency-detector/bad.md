# Notes on Style Consistency

The author *strongly* prefers asterisks for emphasis throughout the doc.

But later, _another section_ slips into underscores.

A mixed paragraph: this is *fine*, this is _also fine_, and this is *fine again*.

Bold like **this** and __that__ are out of scope (handled elsewhere).

```text
Inside a fence: *not* counted, _not_ counted.
```

Inline `*code*` and `_code_` are also ignored.

Closing line with one more *italic*.
