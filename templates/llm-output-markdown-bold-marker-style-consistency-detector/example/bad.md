# Mixed-bold sample

This document mostly uses **double-asterisk bold**, the more common style.

Some more **bold here** and **bold there** for good measure, plus another
**fourth bold** to make the majority unambiguous.

But then it slips and uses __underscore bold__ once.

And again here: __another underscore bold__ on a later line.

Inline code with `**not bold**` should be ignored.

```
**also not bold** inside a fenced code block
```

Final paragraph: still **double-asterisk** as the house style.
