# Examples — Conservative Implementer profile

Two artifacts:

- `transcript-prevented-bad-refactor.md` — a synthetic but realistic mission transcript showing the profile causing the agent to stop before doing damage on a "just add a feature flag" request that was about to drag in a 600-line refactor.
- `compare.md` — same task, run twice: once under this profile, once under a hypothetical "aggressive implementer" profile. Side-by-side outputs with commentary.

These are illustrative, not captured from a single live session, but they reflect actual failure modes seen across many sessions when conservative defaults are absent.
