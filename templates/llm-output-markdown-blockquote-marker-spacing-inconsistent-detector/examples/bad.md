# Mixed blockquote spacing styles

> This file has consistent style for the first quote.
> Two lines, both `> ` (one space).

Then a different quote starts using a different style:

>This one drops the space.
>So does this one.

And later, double-space style sneaks in:

>  This is two-space style.
> But this one is back to one space.
>And this one is no-space.

Even nested ones:

> > nested with one-space
> >nested no-space inner
